# 🛒 ShopHub - Full Stack E-Commerce Website

## 📌 Project Overview

ShopHub is a full-stack e-commerce web application designed to provide users with a seamless online shopping experience. The platform allows customers to browse products, view product details, manage their shopping cart, and place orders efficiently.

This project was built to gain practical experience in frontend development, backend integration, database management, and user authentication while simulating a real-world online shopping platform.

## 🏃‍♂️ Getting Started

### Clone the Repository

```bash
git clone https://github.com/Thanujaroyal24/E-commerce-Website-shophub-.git
```

### Navigate to Project Folder

```bash
cd E-commerce-Website-shophub-
```

### Install Dependencies

```bash
npm install
```

### Run the Application

```bash
npm run dev
```
